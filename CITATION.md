# How to Cite CSA

When referencing Cybernetic Systems Architecture (CSA),
cite the canonical DOI-archived record.

## Canonical citation
Stephanie Luciano Perez.
Cybernetic Systems Architecture (CSA): Canonical Foundations v1.0.
Zenodo. https://doi.org/10.5281/zenodo.18058175

## Notes
- Do not cite GitHub URLs as canonical sources.
- GitHub may be referenced as a mirror or version history only.
- Non-canonical documents must not be cited as standards or authority.
